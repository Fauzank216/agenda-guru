Aplikasi Agenda Mengajar Guru

Tentang Project ini
Aplikasi Agenda Mengajar Guru adalah platform berbasis web yang dirancang untuk mendigitalisasi pencatatan jurnal mengajar harian guru. 
Sistem ini mentransformasikan proses administrasi konvensional yang semula berbasis kertas (paper-based) menjadi sistem digital yang terintegrasi, aman, dan mudah diakses.

Proyek ini awalnya dikembangkan sebagai Proyek Akhir Tahun Pembelajaran (Kelas XI) di sekolah. 
Saya kemudian berinisiatif mengembangkan proyek ini lebih lanjut secara mandiri untuk memperdalam kemampuan, logika berpikir, dan pemahaman saya di bidang web development, khususnya pada ekosistem JavaScript Full-Stack.
Melalui project ini, berikut adalah beberapa target yang ingin saya capai :

1. Pengembangan RESTful API
   - Arsitektur Endpoint Semantik: Merancang rute API terstruktur menggunakan metode HTTP (GET, POST, PUT, DELETE) sesuai standar REST.
   
   - Pertukaran Data JSON: Mengelola komunikasi data dua arah antara backend dan client-side secara asinkronus menggunakan format JSON.
   
   - Operasi CRUD Robust: Membangun logika manipulasi data agenda guru dan kehadiran siswa secara aman dan tervalidasi di sisi server.
   
   - Routing Modular: Mengorganisasikan kode backend menggunakan Express Router untuk menjaga struktur repositori tetap bersih dan terukur.
   
   - Middleware & Validasi: Mengimplementasikan lapisan middleware untuk penyaringan data masuk sebelum diproses oleh fungsi pengontrol (controller).
   
   - Mekanisme Error Handling: Menyusun sistem penanganan error terpusat di sisi server guna mencegah aplikasi crash dan menjaga stabilitas API.
2. Meningkatkan pemahaman mengenai DOM
3. Normalisasi Database : Menerapkan teknik normalisasi untuk menghindari redudansi data
4. Melatih pemahaman query sql: Menggunakan raw query dengan tujuan melatih pemahaman sintaks sql
   
✨ Feature Aplikasi

```
 - CRUD data Master : Guru, Siswa, Kelas, Jurusan, Mata pelajaran, Periode akademik
```
